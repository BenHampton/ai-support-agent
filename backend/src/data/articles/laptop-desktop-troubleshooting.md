## Before You Contact Support

Run the built-in **Ark Diagnostics** tool:

1. Press `F12` during boot
2. Select **Run Diagnostics**
3. The tool checks memory, storage, display, and thermal sensors
4. Share the generated diagnostic report code with our support team

## Boot Failures

**Device does not power on:**
1. Confirm the AC adapter is fully seated
2. Perform a hard reset: hold the power button for 15 seconds, release, then press once

**Device powers on but fails to reach the OS:**
1. Press `F8` during startup to enter **Ark Recovery**
2. Use system restore to roll back to a previous state, or select factory reset
3. Factory reset erases all user data — back up important files before proceeding

> ArkBook Pro devices enrolled in corporate MDM require IT administrator approval before factory reset.

## Display Issues

For dead pixels, backlight failure, or screen artifacts:

1. Run the display diagnostic from Ark Diagnostics
2. If confirmed as a hardware defect and the device is within warranty, file a warranty claim for depot or on-site repair
3. Out-of-warranty screen replacement is available through the Ark Systems Repair Centre — use the Repair Estimator at arksystems.com/repair for pricing

> Liquid damage and physical cracks are not covered under standard warranty.
